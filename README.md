# Christmas List App 2025

## Overview
A web application called the Christmas List App was created to help users manage and arrange ideas for holiday gifts. Users may view gift links, sign in with Google login, and use Firebase Firestore to instantly mark products as purchased.

## Items Used
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firestore Database
- GitHub Pages
- Netlify
